# AdStudio Premium Reklam Üretici Modülü — Implementation Plan

> **For agentic workers:** Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a premium in-app Ad Studio module that can generate professional ad content (storyboard, script, social media post, email) for every module/studio in Oogmatik, using the platform's own AI.

**Architecture:** Types → Service (AI prompt engine via geminiClient) → Hooks (state management) → Components (17 premium features) → Integration (AdminDashboard tab). All data flows through typed interfaces with Zustand-like state management via React hooks + localStorage for persistence.

**Tech Stack:** React 18 + TypeScript (strict) + Vite + Vercel Serverless + Gemini 2.5 Flash + Vitest

## Task 2: Service — `src/services/adGeneratorService.ts`

- [ ] **Create `src/services/adGeneratorService.ts`**

Uses `geminiClient.ts`'s `creativeMultimodalGenerate` to generate ad content. Contains:
- `generateAd(settings, brandKit)` — main generation function
- `generateBatch(targets, settings)` — batch generation
- `generateABVariation(adId, settings)` — A/B test variation
- `buildPrompt(settings, brandKit)` — builds the Gemini prompt

Prompt structure (3 katmanlı):
1. System: Reklam yazarı kimliği + Oogmatik marka bilgisi
2. Context: Hedef modül açıklaması + hedef kitle profili
3. Output: JSON schema zorlaması

```typescript
import { creativeMultimodalGenerate } from './geminiClient';
import { AdStudioSettings, BrandKit, AdOutput, AdScene, AdStudioTarget } from '../types/adStudio';
import { AppError } from '../utils/AppError';

const MODULE_DESCRIPTIONS: Record<AdStudioTarget, string> = {
  math_studio: 'Disleksi ve diskalkuli dostu matematik çalışma kağıtları üretir.',
  reading_studio: 'Okuma güçlüğü çeken çocuklar için hece, kelime ve okuma anlama aktiviteleri.',
  screening_assessment: 'Öğrenme güçlüğü tarama ve bilişsel değerlendirme test bataryası.',
  // ... tüm modüller için
  all_modules: 'Oogmatik platformunun tüm özellikleri.',
};

function buildPrompt(settings: AdStudioSettings, brandKit: BrandKit): string {
  const moduleDesc = MODULE_DESCRIPTIONS[settings.target] || 'Oogmatik eğitim platformu özelliği.';

  return `
BIR OOGMATIK REKLAM METNI YAZ.

HEDEF MODÜL: ${settings.target}
MODÜL AÇIKLAMASI: ${moduleDesc}
HEDEF KITLE: ${settings.audience.join(', ')}
TON: ${settings.tone} (Karışım: ${JSON.stringify(settings.toneMix)})
FORMAT: ${settings.format}
SÜRE: ${settings.duration} saniye
DİL: ${settings.language === 'tr' ? 'Türkçe' : 'İngilizce'}
ÇAĞRI METNİ: ${settings.callToAction}

MARKA:
- İsim: Oogmatik
- Slogan: ${brandKit.slogan}
- Renkler: ${brandKit.primaryColor}, ${brandKit.secondaryColor}

JSON ÇIKTI ŞEMASI:
{
  "scenes": [{ "sceneNo": 1, "duration": 5, "visualDesc": "...", "voiceover": "...", "textOverlay": "...", "transition": "cut" }],
  "script": "tam video senaryo metni",
  "socialCopy": "sosyal medya post metni (280 karakter)",
  "emailSubject": "e-posta konusu",
  "emailBody": "e-posta gövdesi",
  "title": "reklam başlığı"
}

KURALLAR:
1. Tanı koyucu dil kullanma ("disleksisi var" YERİNE "disleksi desteğine ihtiyacı var")
2. Okul yöneticilerine hitap ederken kurumsal faydayı vurgula
3. Her sahne için detaylı görsel açıklaması yaz
4. Süre hesaplaması: voiceover hızı = saniyede 3 kelime
`;
}

export async function generateAd(settings: AdStudioSettings, brandKit: BrandKit): Promise<AdOutput> {
  const prompt = buildPrompt(settings, brandKit);

  const result = await creativeMultimodalGenerate({
    prompt,
    temperature: 0.7,
    systemInstruction: `Sen Oogmatik'in kıdemli reklam yazarısın. ${settings.format} formatında, ${settings.audience.join(' ve ')} hedef kitlesine hitap eden profesyonel reklam içeriği üretiyorsun.`,
  });

  if (!result || !result.content) {
    throw new AppError({
      message: 'Reklam içeriği üretilemedi',
      code: 'AD_GENERATION_FAILED',
    });
  }

  const parsed = typeof result.content === 'string' ? JSON.parse(result.content) : result.content;

  return {
    id: crypto.randomUUID(),
    campaignId: '',
    title: parsed.title || `${settings.target} - ${settings.format}`,
    target: settings.target,
    audience: settings.audience,
    tone: settings.tone,
    format: settings.format,
    duration: settings.duration,
    language: settings.language,
    brandKitId: settings.brandKitId,
    scenes: parsed.scenes || [],
    script: parsed.script || '',
    socialCopy: parsed.socialCopy || '',
    emailSubject: parsed.emailSubject || '',
    emailBody: parsed.emailBody || '',
    tags: settings.tags,
    status: 'generated',
    createdAt: new Date().toISOString(),
    updatedAt: new Date().toISOString(),
    version: 1,
  };
}
```
