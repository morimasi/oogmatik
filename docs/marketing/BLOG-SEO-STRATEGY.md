## 1. CONTENT STRATEGY OVERVIEW

### Content Pillars (Oran)

| Pillar | Oran | Örnek Konular | Frekans |
|--------|------|---------------|---------|
| **Eğitim (40%)** | 40% | Disleksi, özel eğitim, pedagoji | 2/ay |
| **Ürün (30%)** | 30% | Feature guides, how-to, updates | 1-2/ay |
| **Pazarlama (20%)** | 20% | Case studies, success stories | 1/ay |
| **Düşük İçerik (10%)** | 10% | News, industry trends | 1/ay |

### Hedef Kitle Segmentleri

1. **Öğretmenler** (Primary)
   - Keywords: "disleksi materyal", "özel eğitim etkinlik"
   - Amaç: Pedagogical value, time-saving
   - Format: How-to guides, templates

2. **Terapistler** (Secondary)
   - Keywords: "konuşma terapisi etkinlik", "dil gelişim"
   - Amaç: Clinical validation, evidence-based
   - Format: Case studies, research summaries

3. **Veliler** (Tertiary)
   - Keywords: "çocuğumun disleksisi var", "evde nasıl yardım"
   - Amaç: Support, education, resources
   - Format: Explainers, tips, resources

4. **Okul Yönetimi** (Niche)
   - Keywords: "özel eğitim altyapısı", "BEP yazımı"
   - Amaç: Systems, compliance, integration
   - Format: Whitepapers, case studies

## 4. BLOG POST ŞABLONLARI

### Şablon 1: Educational Explainer (2500-3000 kelime)

```
# Disleksi Nedir? Aileler İçin Kapsamlı Rehber

## Introduction (150 kelime)
- Hook: Yüzde istatistiği
- Problem statement: Disleksi tanısı konan çocuklar
- Value proposition: Bu rehber ne sunacak
- Target: Veliler, eğitimciler

## Section 1: Disleksi Tanımı (400 kelime)
- Tıbbi tanım
- Beyindeki mekanizm
- Yaygın yanlış anlamalar

## Section 2: Belirtiler ve Tanı (400 kelime)
- Okul öncesi belirtileri
- Okul dönemi belirtileri
- Tanı yöntemleri
- Teste yönlendirme

## Section 3: Etkileri (300 kelime)

- Duygusal etki
- Sosyal etki
- Kariyer etkileri

## Section 4: Ev ve Okuldaki Destek (600 kelime)

- Ev desteği yöntemleri
- Teknoloji araçları
- Oogmatik çözümü (CTA)

## Section 5: FAQ (300 kelime)
- "Disleksia geçer mi?"
- "IQ etkilenir mi?"
- "Başarılı olamaz mı?"

## Conclusion (200 kelime)
- Key takeaways
- Call-to-action
- Resources

## SEO Elements
- Meta Title (60 char): "Disleksi Nedir? Aileler İçin 2026 Rehberi"

- Images: 4-5 (disleksi diagram, örnek materyaller)
```

## 5. SEO OPTIMIZATION CHECKLIST

### On-Page SEO (Her Blog Post İçin)

- [ ] **Title Tag** (50-60 char, primary keyword)
- [ ] **Meta Description** (150-160 char, compelling)
- [ ] **H1 Tag** (1 per page, keyword included)
- [ ] **H2-H3 Tags** (Logical hierarchy, keywords)
- [ ] **Keyword Density** (Primary: 1-2%, Long-tail: natural)
- [ ] **Internal Links** (3-5, relevant anchor text)
- [ ] **External Links** (2-3, authority sources)
- [ ] **Image Alt Text** (Descriptive, keyword if natural)
- [ ] **URL Slug** (Keyword, hyphens, lowercase)
- [ ] **Readability** (Short paragraphs, lists, bolding)
- [ ] **Content Length** (2000+ words for pillar content)
- [ ] **Mobile Optimization** (Responsive, fast loading)

### Technical SEO

- [ ] **Site Speed** (<2s load time, Google PageSpeed 90+)
- [ ] **Mobile Responsiveness** (100% mobile-friendly)
- [ ] **Schema Markup** (Article, BreadcrumbList, FAQ)
- [ ] **XML Sitemap** (Updated, submitted to GSC)
- [ ] **Robots.txt** (Configured for crawlability)
- [ ] **SSL Certificate** (HTTPS enabled)
- [ ] **404 Handling** (Proper redirects)
- [ ] **Structured Data** (JSON-LD for rich snippets)

### Off-Page SEO

- [ ] **Backlink Strategy** (Target 50+ in year 1)
  - Eğitim siteleri (8-10)
  - Rehber sayfaları (5-10)
  - Haber siteleri (10-15)
  - Akademik siteler (5-10)
  - Sosyal ağlar (10+)

- [ ] **Social Signals**
  - Twitter: Daily posts + engagement
  - LinkedIn: Weekly insights

  - Facebook: Community building

- [ ] **Brand Mentions** (Monitor, respond to all)

## 7. PERFORMANCE METRICS & KPIs

### Monthly Targets

| Metrik | Target | Tracking Tool |
|--------|--------|---------------|
| **Blog Traffic** | 10,000+ views | Google Analytics 4 |
| **Organic Visitors** | 5,000+ | GA4 Source/Medium |
| **Time on Page** | 3+ min average | GA4 Behavior |
| **Scroll Depth** | 50%+ | GA4 Custom Events |
| **Bounce Rate** | <50% | GA4 Engagement |
| **Conversion Rate** | 2-5% | GA4 Goal Completion |
| **Social Followers** | +500/month | Social platforms |
| **Backlinks** | 5-10/month | Ahrefs, SEMrush |
| **Email Subscribers** | +200/month | Email platform |
| **Video Views** | 1,000+ | YouTube Analytics |

### Content Performance Scoring

```
Score = (Traffic × 0.4) + (Engagement × 0.3) + (Conversions × 0.3)

Example:
- Traffic: 2,000 views = 8/10
- Engagement: 3.5 min avg = 7/10
- Conversions: 3% = 8/10
= (8×0.4) + (7×0.3) + (8×0.3) = 7.7/10
```
